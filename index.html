Universal Launcher

Create fancy UIs from simple scripts written in any language.

Get Started
View on GitHub

Supports all platforms

Sunbeam can run on Windows, Linux and MacOS. You can either run it from a terminal, or use the GUI.

Intuitive API, powerful UIs

Sunbeam extensions are composed of a succession of forms, lists, details... But you don't have to write any HTML or CSS.

A flexible extension system based on git

Extensions can be written in any language, as long as they can output JSON. Any repository can be installed as an extension.

Runs as a headless server

Sunbeam can run as an http server, allowing you to access it from anywhere. You can even built your own UI on top of it.
